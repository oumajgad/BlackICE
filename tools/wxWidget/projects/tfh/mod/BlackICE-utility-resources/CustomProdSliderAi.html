On this page you can customize the automatic production slider logic.
If you want to use it you will have to toggle it here and
activate the "Prioritize Upgrades/Custom" mode in the production window.
If you make any change you need to press the "set values" button.

Priority

  • when allocating IC the AI will begin with the priority 1 category
  • each category has to have a unique priority
  • if there is not enough IC available for a category it will allocate all of the remaining IC
  • following categories will have 0 IC assigned

Amount and Mode

  • the amount can be either interpreted as an IC value or as a percentage of the needed IC
  • if there is IC left after all assignments it will be go to production

Extra options

  • reinforcement and upgrades can be limited to prevent them from taking too much IC during extreme situations
  • supply can have a goal amount set in the last column, it will assign a little extra IC until it is reached
  • consumer goods can be enabled to assign 50% extra IC as when you have dissent
